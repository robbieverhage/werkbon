E-mail opstellen
Eenmalige instelling — Google OAuth
1
Ga naar console.cloud.google.com en maak een nieuw project aan (bijv. "Werkbon App").
2
Ga naar APIs & Services → Library en zoek op Gmail API. Klik op Enable.
3
Ga naar APIs & Services → OAuth consent screen. Kies External, vul een appnaam in (bijv. "Werkbon"), sla op. Voeg bij Test users je eigen Gmail-adres toe.
4
Ga naar APIs & Services → Credentials → Create Credentials → OAuth Client ID. Kies type Web application. Voeg bij Authorized JavaScript origins toe: null (voor lokaal gebruik) én het adres waarop je de app opent. Klik op Create.
5
5
Kopieer de Client ID (eindigt op .apps.googleusercontent.com) en plak die hierboven in het veld. Klik op Opslaan & inloggen.
